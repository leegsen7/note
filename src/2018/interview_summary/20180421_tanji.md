### 探迹

问题1：CSS实现水平垂直居中的几种方法
答案如下：
1. flex
```css
.parent {
    display: flex;
    align-items: center;
    justify-content: center;
}
```
2. translate
```css
.child {
    position: relative;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}
```
3. absolute和margin
```css
.parent {
    position: relative;
}
.child {
    position: absolute;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
    margin: auto;
}
```
4. tabal-cell和vertical-align
```css
.parent {
    display: table-cell;
    vertical-align: middle;
}
.child {
    margin: 0 auto;
}
```
